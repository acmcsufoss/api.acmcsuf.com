# api.acmcsuf.com

ACM at CSUF club API for managing events, announcements, forms, and other services!

## Develop

### Start API server

```sh
go run cmd/api/main.go
```

### Generate code

```sh
go generate ./...
```

### Run tests

```sh
go test ./...
```

### Format code

```sh
go fmt ./...
```
